Methodology: Two-Dimensional Classification

Intent Levels (1–4)

1 — Exploring investigating, piloting

2 — Committing investing, building

3 — Deploying launched, operational

4 — Scaling enterprise-wide, expanding

Application Types (6)

  • GenAI / LLMs
  • Predictive ML
  • NLP / Text Processing
  • Computer Vision
  • RPA / Automation
  • Fraud / Risk Models

Classification Engine: Qwen LLM

Why an LLM?

  • Keyword matching can't distinguish "we are exploring AI" from "we scaled AI across all divisions"
  • Context and nuance matter — the same word means different things in different sentences
  • Need structured output: intent level + application categories + confidence for every chunk

Why Qwen?

  • Open-weight model — runs on Colab GPUs, no API cost at scale
  • Strong structured-output capability (returns valid JSON)
  • Fallback chain: 7B → 3B → 1.5B → 0.5B based on available GPU memory

How It Works

Each AI chunk is sent to Qwen with:

  • The chunk text and its source metadata
  • Intent level definitions (1–4)
  • The 6 application categories

Qwen returns a JSON object:

{ "intent_level": 3, "intent_label": "Deploying",
  "app_categories": ["GenAI / LLMs", "Fraud / Risk Models"],
  "confidence": 0.85, "evidence_snippet": "..." }

Fail-Fast Design

If Qwen returns malformed output, the pipeline fails loudly — no silent fallback to weaker heuristics. This guarantees every classification in the final dataset came from the model.

Scoring: Three Dimensions of AI Adoption

Maturity (0–100)
avg(intent levels) / 4 × 100

How far along the explore–to–scale spectrum. A bank averaging level 3 (Deploying) scores 75.

Breadth (0–100)
Shannon entropy / log ₂(6) × 100

How diversified across 6 categories. Equal spread = 100, single category = 0. Penalizes concentration.

Momentum (-100 to +100)
OLS slope × 4 quarters

Is maturity accelerating or decelerating? Regression through quarterly scores, annualized.

Why These Three?

Maturity alone rewards banks that talk big. Breadth catches one-trick ponies. Momentum separates banks that are trending up from those coasting on past deployments.

Composite Score: Putting It Together

Composite = 0.50 × Maturity + 0.35 × Breadth + 0.15 × Momentumnorm

Momentum is normalized from [-100, +100] to [0, 100] before weighting:
Momentumnorm = (Momentum + 100) / 2
